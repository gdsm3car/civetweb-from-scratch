# civetweb-from-scratch

> 采用 Claude Code 辅助，从零复现嵌入式 Web 服务器 [CivetWeb](https://github.com/civetweb/civetweb) 的学习项目。

## 📖 项目简介

本项目的目标是从零开始递进式复现 CivetWeb 的核心功能，深入理解 HTTP 服务端编程、网络通信与 C 语言库嵌入设计。

## 🧭 学习路线

```
里程碑 00 ── 编译 CivetWeb，观察行为
里程碑 01 ── TCP socket 编程基础
里程碑 02 ── HTTP 协议解析
里程碑 03 ── 静态文件服务
里程碑 04 ── 多线程并发（线程池模型）
里程碑 05 ── 回调机制
里程碑 06 ── HTTPS/TLS
里程碑 07 ── WebSocket
里程碑 08+ ── 进阶功能
```

## 🛠️ 方法论

基于 **SETUP 工作流**（分析 → 计划 → 执行 → 关联 → 日志）：

1. **分析** — 理解要实现的模块的工作原理
2. **计划** — 列出实现步骤，先确认再动手
3. **执行** — 编写代码，每个里程碑独立可编译
4. **关联** — 建立知识点间的交叉引用
5. **日志** — 每次学习重点记录在案

## 📂 目录结构

```
├── wiki/         ← 核心知识库（学习笔记、概念分析）
├── my_impl/      ← 手写复现代码（递进式里程碑）
└── SETUP .md     ← Wiki 维护协议
```
