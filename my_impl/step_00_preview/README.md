# 里程碑 00：预览 CivetWeb

> **目标**：编译 CivetWeb 并在 Linux 上跑起来，观察它的行为。
> **原则**：只看现象，不读源码。把这个当作你要复现的"黑盒"。

## 任务

1. 进入 `/raw/civetweb` 目录
2. 阅读 `docs/Building.md`，了解构建方式
3. 编译 CivetWeb
4. 启动服务，用 `curl` 或浏览器访问它
5. 记录观察结果（端口、响应头、支持的路径...）

## 观察清单

启动后尝试以下操作，把结果记录在本目录的 `notes.md` 中：

- [ ] 访问 `http://localhost:8080` — 看到什么？
- [ ] 访问一个不存在的路径 — 返回什么？
- [ ] 查看响应头（`curl -v`）
- [ ] 尝试 WebSocket（如果有测试页面）
- [ ] 查看支持哪些命令行参数

## 产出

- `notes.md` — 观察记录
- 不写一行代码，先「感知」你要复现的东西长什么样
