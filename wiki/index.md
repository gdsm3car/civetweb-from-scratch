# 项目学习 Wiki — CivetWeb

> **学习目标**：从零开始复现一个嵌入式 C/C++ Web 服务器，深入理解 HTTP 协议、网络编程与库嵌入设计。

## 🗂️ 目录结构

| 路径 | 用途 | 说明 |
|------|------|------|
| `/raw/civetweb` | 原始代码 | CivetWeb 官方仓库（只读，作为对照参考） |
| `/wiki` | 核心知识库 | 学习过程中的知识沉淀 |
| `/my_impl` | 复现代码 | 从零手写，每一步一个可编译的里程碑 |

## 🧭 学习路线图

```
里程碑 0 ── 编译 CivetWeb，观察行为（只跑不读源码）
    │
    ▼
里程碑 1 ── socket 编程基础（TCP server）
    │
    ▼
里程碑 2 ── HTTP 协议解析（读懂请求报文）
    │
    ▼
里程碑 3 ── 静态文件服务（serve 目录）
    │
    ▼
里程碑 4 ── 多线程并发（线程池模型）
    │
    ▼
里程碑 5 ── 回调机制（用户自定义 handler）
    │
    ▼
里程碑 6 ── HTTPS/TLS（加密通信）
    │
    ▼
里程碑 7 ── WebSocket（全双工通信）
    │
    ▼
里程碑 8+ ── 进阶：路由、CGI、Lua、性能优化...
```
